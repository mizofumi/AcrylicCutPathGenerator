# Acrylic Cut Path Generator (Java版)
- Windows, macOS, Linux対応  

Windowsの方は JDK21をインストールの上で jar ファイルをクリックすると起動します。  

macOS,Linuxの方は、JDK21をインストールの上で　jarをクリックするか、以下のコマンドで起動できます。
```
java -jar acrylic-cut-path-generator-1.0.0-all.jar
```

# 【非推奨】 Acrylic Cut Path Generator (Web版)【Deprecated】

![スクリーンショット 2025-06-18 1 40 57](https://github.com/user-attachments/assets/9d1e0deb-9762-4037-b419-297067b78d00)

A web application for generating cutting paths for acrylic materials.

## Prerequisites

- Docker
- Docker Compose

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/mizofumi/AcrylicCutPathGenerator.git
   cd AcrylicCutPathGenerator
   ```

2. Start the application using Docker:
   ```bash
   docker compose up -d
   ```

3. Access the application in your browser:
   [http://localhost:3000](http://localhost:3000)
